# TizWildin Hub JUCE Scaffold

This is a file-level scaffold for the JUCE desktop hub app.

## Expected top-level UI
- sidebar groups
- product grid/cards
- refresh button
- install missing button
- check updates button
- account panel
- seats panel
- purchase/open billing flows

## Integration points
- call ARC service for catalog, entitlements, seats, and proposals
- scan local plugin folders
- map state into card badges and actions
