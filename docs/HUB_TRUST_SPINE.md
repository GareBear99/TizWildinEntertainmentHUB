## Public positioning

The HUB is the public routing layer for Gary Doman / GareBear99 / TizWildin Entertainment projects. It connects free audio plugins, source repositories, release records, update paths, sample packs, public discovery lists, and ARC source-spine routes into one crawlable index.

## What the HUB does

- Routes users to free-version plugin downloads and source repos.
- Checks GitHub releases for public plugin updates.
- Exposes machine-readable manifests and public indexes.
- Gives search engines and AI tools a source-of-truth map of the ecosystem.
- Separates product discovery from checkout/payment logic.

## What the HUB does not claim

- It does not claim to be the checkout.
- It does not claim every plugin has a packaged release yet.
- It does not hide behind fake trust claims.
- It does not replace each product's own documentation, licensing, or release process.

## Why this matters

A polished frontend alone is not enough. A real public ecosystem needs crawlable proof, source routes, manifests, release records, and durable public pages. This HUB keeps the marketing layer and the source-truth layer together.
